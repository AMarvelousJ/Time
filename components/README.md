# 组件目录

## 目录结构

### `ui/`
shadcn/ui 基础组件
- `button.tsx` - 按钮组件
- `dialog.tsx` - 对话框组件
- `input.tsx` - 输入框组件
- `label.tsx` - 标签组件
- `card.tsx` - 卡片组件
- `accordion.tsx` - 手风琴组件（阶段导航）
- `badge.tsx` - 徽章组件（状态标识）
- `progress.tsx` - 进度条组件
- `separator.tsx` - 分隔线组件

### `layout/`
布局组件（待创建）
- `header.tsx` - 页面头部
- `sidebar.tsx` - 侧边栏
- `stage-accordion.tsx` - 阶段手风琴导航

### `forms/`
表单组件（待创建）
- `date-picker.tsx` - 日期选择器
- `time-field.tsx` - 时间字段输入
- `conflict-alert.tsx` - 冲突提示
- `recommendation-tip.tsx` - 推荐提示

### `timeline/`
时间轴组件（待创建）
- `gantt-chart.tsx` - 甘特图
- `timeline-view.tsx` - 时间轴视图

### `store/`
状态感知组件（待创建）
- `person-switcher.tsx` - 人员切换器
- `save-indicator.tsx` - 保存状态指示器
